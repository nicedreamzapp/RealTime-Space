This repo holds both platforms of RealTime Space, cleanly separated:

## 📱 iOS

SwiftUI + `WKWebView` wrapping a Three.js/WebGL galaxy renderer. Full source, Xcode project, and textures live in [`ios/`](ios/).

**Status:** ✅ **Live on the App Store** — [download it here](https://apps.apple.com/us/app/realtime-space/id6788646103) (free).

## 🤖 Android

The Android port — a WebView shell around the same renderer, with all web assets bundled. Source in [`android/`](android/).

**Status:** Submitted to Google Play — review in progress (July 2026). The code is available now in [`android/`](android/).
